-- SUMMARY --

The Meeting Update module adds a Date and Time field to existing Committee Meeting nodes.

-- INSTALLATION --

The module installs just as any other Drupal module.

-- CONFIGURATION --

None needed.

-- USING THE MODULE --

Simply press the Submit button.

This is a purpose-built module that will loop through Committee Meeting nodes and overwrite whatever is in the "Date and Time" field with 

Submit the form. It will take a long time.

When done, you will see a Drupal message to that effect, along with the number of nodes changed and the total nodes.
